# Entries
## Projektről

BME-s és SCH-s email címmel ingyenesen elérhető prémium szolgáltatásokat nyílvántartó program. 

## Függöségek
- python3
- django == 1.9.5
- django-rosetta
- pillow
- bower
- jquery
- uikit
- sass

## Telepítés

````pip install django==1.9.5````

````pip install pillow````

````bower install````

````sassc sass/base.sass sass/base.css````

````./manage.py migrate````

````./manage.py runserver````
